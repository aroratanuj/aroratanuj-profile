# 🚀 Deploy Your Portfolio Publicly

## Option 1: Netlify (Easiest - Drag & Drop)

### Steps:
1. Go to [netlify.com](https://www.netlify.com/)
2. Sign up/login (free)
3. Drag & drop your `aroratanuj-profile` folder onto the Netlify dashboard
4. Your site will be live instantly at: `https://random-name.netlify.app`

### Pros:
- ✅ Super easy (drag & drop)
- ✅ Free SSL certificate
- ✅ Custom domain support
- ✅ Automatic deployments
- ✅ Fast CDN

---

## Option 2: GitHub Pages (Best for Developers)

### Steps:
1. Create a new GitHub repository
2. Upload your `index.html` file
3. Go to Settings → Pages
4. Select source: "Deploy from a branch"
5. Select branch: "main" or "master"
6. Your site will be at: `https://yourusername.github.io/repository-name`

### Or use command line:
```bash
cd "/Users/tanuj/Desktop/AI Folder/aroratanuj-profile"

# Initialize git repo
git init

# Add files
git add .

# Commit
git commit -m "Initial portfolio commit"

# Create GitHub repo first, then:
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git push -u origin main
```

### Pros:
- ✅ Free forever
- ✅ Version control
- ✅ Custom domain
- ✅ Great for developers

## Option 3: Vercel (Modern & Fast)

### Steps:
1. Go to [vercel.com](https://vercel.com/)
2. Sign up/login with GitHub
3. Click "New Project"
4. Import your repository or drag & drop folder
5. Deploy!

### Pros:
- ✅ Extremely fast
- ✅ Great developer experience
- ✅ Automatic deployments from Git
- ✅ Free custom domains

---

## Option 4: Cloudflare Pages (Free & Fast)

### Steps:
1. Go to [pages.cloudflare.com](https://pages.cloudflare.com/)
2. Sign up/login
3. Click "Create a project"
4. Connect to GitHub or upload assets
5. Deploy!

### Pros:
- ✅ Free SSL
- ✅ Unlimited bandwidth
- ✅ DDoS protection
- ✅ Fast global CDN
